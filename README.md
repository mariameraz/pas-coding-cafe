# PAS Bioinformatics & Coding Café (Spring 2026)

Casual talks + hands-on sessions for the PAS (Plant & Agroecosystem Sciences) community at UW, focused on computing skills, coding practices, data analysis, and practical bioinformatics tools.

**When/Where:** Spring 2026 - Moore Hall  
**Format:** ~1 hour per session (tutorial + discussion + help on your own projects)  
**Open to everyone:** all backgrounds and experience levels welcome.

## Goals
- Share tutorials, workflows, and tools
- Improve research communication skills
- Promote reproducible and open science
- Build a supportive student community to help each other with coding and project questions

## What’s in this repo
- `sessions/` — slides, notes, and resources per session
- `workshops/` — hands-on materials (datasets, notebooks, exercises)
- `examples/` — small example projects and reproducible workflows
- `resoruces/` - libraries, tools, and learrning resources
- `CODE_OF_CONDUCT.md` — community guidelines
